函数式数据处理

函数式数据处理

1. 流的概念

集合:用来存储数据的容器

流(Stream): 用来处理数据的工具

优势:

  1. 简化编程
  2. 提高数据的处理效率

2. 流的运算过程

由数据源放入流中开始，中间操作之后全部是流式处理结果，流处理结果经过终端操作后才能转化为结果数据：

数据源——–>中间操作——–>中间操作——–>终端操作———>结果数据

中间操作: Stream<T>
方法 作用 返回值
filter(Predicate) 过滤 , 筛选 Stream
sorted(Comparator) 排序 Stream
distinct() 去重 Stream
limit(long) 取前几个元素 Stream
skip(int) 跳过前几个元素 Stream
map(Function<T,R>) 将流中的每个元素利用函数映射为另一个元素 Stream
flatMap(Function<T,R>) 将分散的流组合成一个整体的Stream Stream
中间操作示例：

map映射：参数为Function接口，将流中的每个元素利用函数映射为另一个元素

flatMap：将多个T对象转为多个Stream，再做一个归集，将多个Stream归集为一个Stream

终端操作 Stream<T>
方法 作用 返回值
count() 返回元素的个数 long
forEach(Consumer) 遍历Stream中的所有元素 void
max(Comparator) / min(Comparator) 找最大/最小的元素 Optional
findAny() 找到Stream中的某个对象 Optional
findFirst() 找到Stream中的第一个对象 Optional
allMatch(Predicate) 判断流中的元素是否都匹配条件 boolean
anyMatch(Predicate) 判断流中得到元素是否有一个能匹配条件 boolean
noneMatch(Predicate) 判断流中的元素是否都不能匹配条件 boolean
collect(Collector<T,A,R>) 收集流中的元素 R

R collect(Collector) Collector: 收集器 <T,A,R> 将Stream 转化收集最终成为 R对象

Collectors 工具类, 可以得到常用的各种Collector

4.数值流

基本类型的特化

  • IntStream DoubleStream LongStream

  • Stream -> IntStream

    • mapToInt(ToIntFunction) T–>int
  • IntStream -> Stream
    • boxed()
  • 数值范围
    • range(1,100) 左开右闭
    • rangeClosed(1,100) 左闭又闭

1）Stream —> IntStream：来自Stream的mapToInt()方法
